How the light shell works
  • Shared chrome — same step strip and capture pills on every water tool.
  • Project name — enter once in the shell; it follows all four steps and report captures.
  • Independent use — bookmark any step; skip imports and enter numbers manually.
  • Handoffs — Capture / Capture & Open writes the next tool’s input when you need synergy.
  • Reports — Benham, MeadHunt JV, Haskell (by state), or upload — shared across the path and other Toolshed report tools.
