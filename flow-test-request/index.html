Outlet limitation

Stock procedures in this package apply to flowing 2½-inch hydrant outlets only (typical hose outlets). Pumper outlets are intentionally excluded from this request template. If a pumper-outlet test is required by the AHJ or base, note it under jurisdiction requirements and confirm equipment / coefficients separately.
