# Study Buddy

Multi-class study platform (source under `study-buddy/`).

## Share / run on the web (GitHub Pages)

**https://pistolpetefire.github.io/Fire_toolshed/**

- Hub: https://pistolpetefire.github.io/Fire_toolshed/
- Anatomy (BIO 1314 / BIO 1414): https://pistolpetefire.github.io/Fire_toolshed/classes/occc-bio-ap

### Enable once

1. https://github.com/pistolpetefire/Fire_toolshed/settings/pages  
2. **Source** → **GitHub Actions**  
3. Actions tab → workflow **Deploy Study Buddy** should turn green  

## Run locally

```bash
cd study-buddy
npm install
npm run dev
```
