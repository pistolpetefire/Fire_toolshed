Fire Tank Sizer — Help / Input Guide

Purpose: Preliminary fire water tank sizing from system/pump flow × duration, plus planning safety factors. Select a US-typical prefab steel size under site height/diameter limits, with a concrete pad plan. Not a sealed NFPA 22 design.

Path: Flow Test → Sprinkler Estimator → Fire Pump Sizer → Tank Sizer.

Demand inputs

  • System / pump flow (GPM): Usually total fire water demand from the Sprinkler Estimator (or pump duty flow).
  • Duration (min): Water-supply duration from code/framework (UFC Table 9-4, NFPA 13, FM, project). Imported with sprinkler capture when available.
  • Import Upstream Demand: Prefers Fire Pump Sizer capture (flow × duration), else Sprinkler Estimator handoff.
  • Import Pump Demand: Loads Capture for Tank from Fire Pump Sizer.
  • Import Sprinkler: Loads Capture for Pump Sizer (flow + duration + project fields).

Safety framework scenarios

  • NFPA planning (+5% default): Base = Q×t; small contingency for freeboard/suction planning. Confirm NFPA 22 effective capacity.
  • UFC planning (+10%): Higher contingency for DoD-style projects; confirm UFC duration/redundancy.
  • FM Global planning (+15%): More conservative planning; cite governing FM DS.
  • Safety contingency override %: Blank = use framework default; enter a number to force a custom %.
  • Unusable %: Increases required shell volume so usable water still meets demand after freeboard/suction allowances.

Site constraints

  • Max tank height: Local zoning / AHJ / aircraft / aesthetic limit. Filters prefab catalog. 0 = no limit.
  • Max diameter: Site footprint / setbacks. Filters catalog. 0 = no limit.
  • Pad beyond shell: Concrete pad extends this far past tank diameter on each side (plan diameter = tank ⌀ + 2× beyond).
  • Pad thickness: Used only for approximate concrete volume (cu yd). Structural design by others.

Prefab catalog & custom

  • Rows are typical US bolted/reservoir/standpipe-style steel fire tanks (planning nominals).
  • Green/white rows meet volume + max H + max ⌀. Click to select.
  • Custom diameter & height: Override catalog with project-specific geometry; capacity from cylinder volume.

Outputs

  • Plan diagram: Tank + concrete pad with dimensions (pump-sizer style).
  • Clean Report: On-screen preview + print dialog.
  • Save Report: Downloads HTML report + JSON data files.
  • Save tank handoff: Stores selection for future tools / records in browser storage.

Scenarios

  • Height-limited site: Set max H low; catalog prefers wider, shorter tanks.
  • Tight footprint: Set max ⌀; taller options appear if height allows.
  • Long duration (UFC extra / FM): Import duration from sprinkler app; required shell grows with Q×t and safety %.
