Sprinkler Estimator — README / Input Guide

Purpose: Preliminary / charette sprinkler demand + pressure stack for AHJ discussion. Not a final Hazen-Williams calc.

Path: Flow Test → Sprinkler Estimator → Fire Pump Sizer → Tank Sizer.

Save Report downloads an HTML report and a JSON project snapshot. Clean Report opens the print package.

Estimate mode

  • Simple — manual system demand: Enter total GPM when you already have a planning or hydraulic total.
  • Criteria-assisted: Seed density, design area, hose, K, and duration from NFPA 13, UFC 3-600-01, AF Sundown, or FM Global presets, then edit.

Criteria framework

  • NFPA 13: Density-area light / ordinary / extra. Dry uses +30% design area (typical dry/double-interlock rule).
  • UFC 3-600-01: Full Table 9-3 density/area/K by hazard, wet/dry, ceiling height. Hose from Table 9-4. High bay (>60–100 ft) uses 12 sprinklers @ 7 psi, K25.2 (wet only).
  • AF Sundown: Fixed wet criteria 0.20 gpm/sf over 5,000 sf, K8.0, 130 sf/head. No dry path.
  • FM Global: Planning placeholders — cite the actual DS. Storage/ESFR wet only.

System type

  • Wet pipe: Also single-interlock / non-interlock preaction (UFC “wet” column).
  • Dry pipe: Also double-interlock preaction / deluge (UFC “dry” column). Larger design areas where table requires. Not available for AF Sundown or ESFR/storage.

Demand inputs

  • Density / design area: Sprinkler demand = density × area (unless high-bay heads mode).
  • Hose allowance: Inside/outside hose stream GPM added when checkbox is on.
  • K-factor / coverage / remote head flow: Used to compute remote head required pressure P = (Q/K)², floored by code minimum (often 7 psi).
  • Elevation: Static head = feet × 0.433 psi/ft from pump discharge datum to highest head.
  • Backflow / equipment losses: Planning lump-sum pressure losses — not pipe-by-pipe HW.
  • Safety margin: Explicit charette contingency in psi.

Duration required (minutes)

  • UFC Table 9-4: Light & Ordinary = 60 min; Extra = 90 min.
  • NFPA 13 (typical): Light 30; OH-1 60; OH-2 90; EH-1 90; EH-2 120 (confirm adopted edition — some values are ranges).
  • FM Global (planning): Ordinary/general 60; Extra 90; Storage/ESFR 90 (confirm governing DS — may be longer).
  • AF Sundown: 60 min planning default.
  • Uncheck Auto-set duration to enter a project-specific value. Duration × flow ≈ prelim tank gallons.

Water supply — flow test curve

  • Use saved flow test: Loads the last Flow Test Report save (static, residual, flow @ residual) from this browser.
  • Supply curve: P = Pstatic − (Pstatic−Presidual)×(Q/Qtest)1.85 on an N1.85 flow scale.
  • Demand point: This estimator’s system demand flow @ system demand pressure.
  • Graph + table appear on screen and on the Clean Report when enabled.

Outputs & handoff

  • System demand flow / pressure: Prelim duty at pump (for Fire Pump Sizer).
  • Capture for Pump Sizer: Saves flow, pressure, duration, and basis to browser storage and copies a text handoff. Open Fire Pump Sizer → Import Sprinkler Demand.
  • Clean Report: Print-ready charette package with summary, bases, pressure stack, and supply/demand graph when a flow test is loaded.
  • Save Report: Downloads HTML report + JSON project snapshot to your computer.

Limitations

  • No node-by-node Hazen-Williams analysis or remote-area proof.
  • Presets are starting points — cite adopted edition, tables, and AHJ direction.
  • Sloped-ceiling +30% area (UFC when slope > 2:12) is not auto-applied; increase design area manually if required.
