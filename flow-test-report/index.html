    Flow Test Report — Help / Input Guide

    Purpose: Document a hydrant flow test per NFPA 291 guidance, build an N1.85 supply curve, and capture results for the Sprinkler System Estimator. This app does not calculate sprinkler system demand.

    Path: 1 Flow Test → 2 Sprinkler Estimator → 3 Fire Pump Sizer → 4 Tank Sizer.

    Project / test info

    • Project name / site: Appears on reports and the sprinkler handoff label.
    • Test date/time: Used for gauge calibration age vs test date.
    • Tested by / witness: Field documentation for the report package.
    • Test purpose:
      • Fire suppression system water supply — document available supply for design; demand is applied later in Sprinkler Estimator.
      • Fire flow / hydrant capacity rating — emphasize rated capacity at the rating residual (often 20 psi) and related NFPA 291 checks.

    Hydrant layout (Fig. 4.3.4)

    • Residual hydrant (R): Where static and residual pressures are read.
    • Flow hydrants (F): One or more separate hydrants for discharge. Prefer ≥1 flow hydrant; 0 = single-hydrant test (warned).
    • Layout notes: Direction of flow, relative positions on the main.

    Gauge calibration

    • Static/residual gauge: Typically 0–200 psi on residual hydrant. Calibration date checked for ≤12 months (NFPA 291).
    • Pitot gauge: Typically 0–50/60 psi. Same 12-month check.
    • Optional gauge IDs for equipment tracking.

    Hydrant flow test readings

    • Static pressure: No-flow pressure at residual hydrant.
    • Residual pressure: Pressure at residual hydrant while flowing.
    • Flow at residual: Total discharge GPM during the test (manual or from pitot helper).
    • Min residual / rating pressure: Reference for capacity at residual (often 20 psi) and the curve reference line.
    • Pressure drop %: NFPA 291 recommends ≥25% drop from static for satisfactory theoretical results.

    Pitot flow helper

    • Pitot pressure / outlet diameter: Used in Q = 29.83 × C × d² × √P.
    • Outlet geometry: Sets C from NFPA 291 Fig. 4.7.1 (0.90 / 0.80 / 0.70) or flow tube 0.95.
    • Additional open butts: Rough multiplier when multiple outlets share similar readings.
    • Monster Pitot Chart: Forces C = 0.9 and rounds to 5 gpm.

    Procedure checklist & data sheet

    • Checklist tracks NFPA 291 field steps (bleed air, open one-at-a-time, simultaneous read, slow shutdown, valve wide open, pitot position, radios).
    • Data sheet: main size, weather, operators, last/next test, ordinary demand confirmation, notes.

    Outputs

    • Supply curve: N1.85 hydraulic scale; no system demand point here.
    • Capture for Sprinkler Estimator: Saves supply data to browser storage for Step 2. Use Capture & Open Sprinkler for one-click handoff.
    • Multi-outlet pitot: Measure each open butt (or use the rough multiplier). Pumper 4–4½″ options use Table 4.8.2-style C ≈ 0.90.
    • Pitot 10–30 psi: Preferred band (NFPA 291 4.6.6) — out-of-band readings warn on the checklist.
    • Capacity rounding & Class AA–C: Rated capacity rounded (50 gpm <1000, 100 gpm ≥1000); class AA ≥1500, A 1000–1499, B 500–999, C <500 gpm.
    • Save report: Downloads a standalone HTML report file to your computer.
    • Print report: Browser print / PDF of the on-screen report package.

    Scenarios

    • Design water supply for a building: Purpose = suppression supply → capture → Sprinkler Estimator demand + duration → pump/tank.
    • Hydrant capacity marking / ISO-style flow: Purpose = fire flow → review rated capacity at rating residual and drop %.
    • Weak main / small drop: Open more outlets or hydrants until ≥25% drop (or document total demand flowed).
