Notes

  • Preliminary only — confirm IBC/NFPA 101 edition adopted by the AHJ.
  • Doors: use clear width; minimum clear widths and door swings are separate code checks.
  • Stairs: capacity is for the stair width serving the story; stair geometry (riser/tread) not checked here.
  • Multiple stories, horizontal exits, and smokeproof enclosures need additional analysis.
  • Assembly concentrated seating and fixed seating may use seats/count instead of area.
